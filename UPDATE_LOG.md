# UPDATE LOG

## [1.5.0] - 2026-02-13
### 追加 (Added)
- ライセンス申請時に管理用ウェブフックへ通知を送信する機能を追加。
- メッセージ形式の申請と、UI（モーダル）形式の申請の両方に対応。
- 通知内容に申請者名、希望プラン、対象サーバーID、Booth名を含めるように改善。
- 告知作成時の「マイルストーン告知テンプレート機能」を実装。`{{M2}}` などの記述を自動的に詳細内容へ置換可能に。



## [1.4.1.1] - 2026-02-13
### 修正 (Fixed)
- サーバー起動時の `TypeError: argument handler must be a function` エラーを修正。
- ルートの読み込み時に型チェックを追加し、不正なハンドラーによるクラッシュを防止。
- ルーティングエラー発生時の調査を容易にするため、詳細なエラーログ出力を追加。
- `routes/misc.js` のエクスポート処理を整理。
